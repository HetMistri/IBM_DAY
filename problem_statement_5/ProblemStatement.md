🏛️ Problem 5: The Final Dial

The final chamber of the temple reveals a massive stone chest.
At its center lies a circular dial, marked with numbers from 0 to 99.
An arrow points to a number on the dial.
Carved above it:
“You have come far.
 One final sequence remains.
 Follow it precisely… and the treasure is yours.”

🔍 The Dial
The dial contains numbers from 0 to 99
It is circular:
Moving right from 99 takes you to 0
Moving left from 0 takes you to 99
⚙️ The Rotations
Each rotation is written as:
Lx   or   Rx

Where:
L → rotate left (towards smaller numbers)
R → rotate right (towards larger numbers)
x → number of steps

🧭 Movement Rules
Each step moves the dial by 1 position
The dial wraps around at both ends
Each rotation starts from the current position
🧠 Starting Point
37

🔁 Rotations

R18
L42
R77
L15
R64

🎯 The Task
Follow all rotations in order.
After each rotation:
Note the number where the dial stops
At the end:
Compute the sum of all these numbers


🔐 Final Answer
Submit a single integer — the final sum.

“ The chest opens only for those who make no mistakes. ”
